Development Notes
=================

Requirements:
-------------
* node v6.4.0
* npm 3.10.7
* AWS credentials file in '~/.aws/credentials'
```
[default]
aws_access_key_id=***PROTECTED***
aws_secret_access_key=***PROTECTED***
region=eu-central-1

```

Commands
--------
* Serve locally: `npm run serve`
* Compile: `npm run gulp`
* Deploy: `npm run deploy`
